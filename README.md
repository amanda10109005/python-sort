# Bubble Sort
Bubble sort adalah sebuah algoritma dengan pengurutan sederhana yang bekerja dengan membandingkan dan menukar posisi elemen-elemen yang saling berdekatan satu sama lain dalam array secara berulang hingga array terurut.

Bubble sort ascending adalah algoritma pengurutan sederhana yang mengurutkan elemen-elemen dalam array dari kecil ke besar. Algoritma ini membandingkan dan menukar posisi elemen-elemen array yang saling berdekatan satu sama lain yang urutannya salah hingga array terurut secara ascending.

Bubble sort descending adalah algoritma pengurutan sederhana yang mengurutkan elemen-elemen dalam array dari nilai terbesar hingga nilai terkecil. Algoritma ini bekerja dengan membandingkan dan menukar posisi elemen-elemen yang saling berdekatan satu sama lain ini dengan urutannya salah. Proses ini diulang hingga tidak ada lagi pertukaran posisi yang dilakukan, sehingga array menjadi terurut secara descending.

jadi dapat disimulkan bahwa pebedaan terlihat pada urutan pengurutan elemen dalam array. Yang dimana bubble sort ascending mengurtkan dari elemen terkecil, sedangkan bubblesort descending mengurutkan dari yang terbesar

# Selection Sort
Selection sort adalah sebuah algoritma dengan pengurutkan yang secara berulang-ulang untuk mencari elemen terkecil maupun terbesar dari sisa daftar dan menukar posisinya dengan elemen pada indeks terkini.

Selection sort ascending adalah algoritma pengurutan sederhana yang mengurutkan elemen-elemen dalam array dari nilai terkecil hingga nilai terbesar. Algoritma ini bekerja dengan mencari elemen terkecil dari sisa array yang belum terurut dan menukarkannya dengan elemen pertama. Proses ini diulang secara berulang hingga seluruh array terurut secara ascending.

Selection sort descending adalah algoritma pengurutan sederhana yang mengurutkan elemen-elemen dalam array dari nilai terbesar hingga nilai terkecil. Algoritma ini bekerja dengan mencari elemen terbesar dari sisa array yang belum terurut dan menukarkannya dengan elemen pertama. Proses ini diulang secara berulang hingga seluruh array terurut secara descending.

Dari selection sort ascending dan descending, keduanya merupakan sebuah algoritma pengurutan sederhana yang bekerja dengan mencari elemen terkecil maupun terbesar dan menukarnya dengan elemen pertama. Perbedaan antara keduanya terletak pada arah pengurutan yang dihasilkan
